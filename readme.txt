== Description ==

Simple and Easy CMS Block which generates Shortcode and can be reuse in any place of the website. Sometimes a same HTML block needs to display multiple places so this plugin really helps. Write content as CMS Block and paste shortcode where you need to be display.

Are you looking for the best CMS plugin for WordPress? Look no further, as CMS Blocks is a versatile Content Block Wordpress plugin that enables you to create and manage custom content blocks seamlessly. You can create custom blocks, styles, variations, and patterns without requiring a code editor.
Aided with an easy-to-use interface, it allows you to insert custom block shortcodes into your pages and posts. Offering a variety of benefits, CMS Blocks makes for the best WordPress content builder that boosts your website's performance.

<strong>Features:</strong>
<ul>
	<li>Create custom content blocks.</li>
	<li>Easily integrate the Shortcodes whenever required.</li>
	<li>Flexibility of the layout enables versatile usage.</li>
	<li>Personalization to add shortcode in page, post or any other content areas.</li>
	<li>Generate the Shortcode in a simple and effective way.</li>
	<li>Reuse the created content blocks to ensure all-round consistency across your website.</li>
	<li>Create, organize, and edit content blocks with complete flexibility and control over what is being displayed.</li>
	<li>Efficient drag-and-drop interface that aids you in positioning content blocks as needed on your web pages.</li>
</ul>

== Installation==

1. Upload `cms-block` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Manage from CMS Blocks in left menu.

== Frequently Asked Questions ==

= How can I add Blocks in posts or pages? =

You can add Blocks by following the below steps:
<ul>
	<li>Go to “CMS Blocks” in the WordPress admin panel.</li>
	<li>Add content in the editor and save.</li>
	<li>Copy the Shortcode and place it in the page or post editor.</li>
</ul>

= Can I use CMS Blocks for eCommerce purposes? =

Yes. CMS Blocks help you display dynamic content like product features, promotions, and others on your eCommerce websites.

= Do I need coding skills to use this content builder block plugin? =

Using CMS Blocks doesn’t require coding skills, as they are designed to help users with any expertise level. You can simply create content blocks and use them for your pages with ease.
